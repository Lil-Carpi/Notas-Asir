En PHP, las variables se indican con un `$` y luego un nombre. Podemos almacenar tanto strings, booleans y integers dentro para poder utilizarlo más tarde.

```php
<?php
$nombreUsuario = "Cubo";
$edadUsuario = 20;
?>
<h1>"Hola <?php$nombreUsuario;?>, tienes <?php$edadUsuario;?> años."</h1>
```
>[!Output]
>Hola Cubo, tienes 20 años
