## Personalización
También, deberás de buscar un tema de tu agrado [aquí](https://sphinx-themes.org/).
Para instalar el tema, haz:
```python
pip install shibuya
```

Una vez que instales el tema, edita el fichero `conf.py` de la documentación de Sphinx y busca `html_theme`:
```python
html_theme = 'shibuya'
```
