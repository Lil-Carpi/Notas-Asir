SELinux es un sistema de segurización incluido en el paquete de seguridad para el [[Kernel Linux|kernel de Linux]] en el framework del núcleo *Linux Security Modules* creado por Red Hat y la NSA en el 22 de noviembre del 2000. Fue creado en [[C - Índice|C]], y proporciona el mecanismo para implementar [[Políticas de control de acceso]] y de tipo *[[Control de Acceso Obligatorio (MAC)]]* y control de acceso basado en roles.

Está disponible en las distribuciones de [[Fedora]], [[Red Hat]], [[CentOS]], [[Scientific Linux]] y Android. En estas distribuciones, está activo por defecto.

Se puede configurar SELinux mediante la consola web [[Cockpit - Fedora Server|Cockpit]], solo disponible en la distribución de servidor de [[Fedora Server|Fedora]].
