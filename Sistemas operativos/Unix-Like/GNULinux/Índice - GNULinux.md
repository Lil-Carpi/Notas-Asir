GNU/Linux es EL sistema operativo. Pertenece a la familia de sistemas operativos tipo Unix compuesto por software libre y de código abierto.
GNU/Linux surge de las contribuciones de varios proyectos de software, GNU (Iniciado por la leyenda Richard Stallman en 1983) y el núcleo Linux (hecho por Linus Torvalds en 1991).

A pesar de que al sistema operativo en sí se le llame solo *Linux*, este es solo el [[Kernel]]. El sistema operativo completo incluye el sistema GNU.
