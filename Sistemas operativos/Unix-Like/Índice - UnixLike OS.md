#UnixLike - Sistemas operativos tipo Unix
Documento padre
---
Un sistema operativo tipo UNIX, que en algunos casos se abrevia como UN#X, o NIX para no tener problemas con la marca registrada (UNIX es privativo), es un sistema que se comporta de manera similar a UNIX, aunque no necesariamente ha de estar certificado en ninguna versión de Single Unix Specification.
Realmente no hay una norma para definir el término *UNIX-Like*, por lo que se generaron diferentes opiniones sobre cuando llamar a un sistema operativo "UNIX-like".
