El **WWW** (acronimo ingles de World Wide Web, Red de Abasto Mundial) o **web** es una red de paginas escritas en hipertexto mediante el lenguaje de marcado [[Índice - HTML|HTML]] y conectados entre si mediante vinculos, de manera que formen un solo cuerpo de conocimiento por el que se pueda manejar facilmente. Para acceder a el es indispensable un [[Navegador Web]]. Se creo por [[Tim Berners-Lee]] cuando trabajaba en el CERN de Ginebra, en Suiza. Él mismo dirige el W3C, organismo encargado de mantener el funcionamiento.

La web se basa en tres estandares para funcionar: el [[Uniform Resource Locator]] (URL), que se encarga de dar una direccion unica para la localizacion de cada pagina; el Hyper Text Transfer Protocol ([[HTTP - Protocolos de red|HTTP]]), que especifica la manera en como se enviara y recibira la informacion entre el navegador y el servidor; Hyper Text Markup Language ([[Índice - HTML|HTML]]), un metodo para especificar como se tiene que ver la informacion ene le navegador. Acompaña al HTML el [[Índice - CSS|CSS]], para definir aspectos de diseño, y [[Índice -JavaScript|JavaScript]] para hacer pequeñas programaciones dentro de la pagina web.

### Historia
El 13 de marzo del 1989, Tim Berners-Lee, que trabajaba en el CERN de Ginebra des del 1984 en tiempos de adquisicion y trata de datos, se propuso un sistema de hipertexto organizado en web con tal de mejorar el intercambio de informacion entre sus investigadores (*[Information Management: A Proposal](https://www.w3.org/History/1989/proposal-msw.html)*) basado en un proyecto llamado ENQUIRE que habia hecho en el 1980. Con ayuda de [[Robert Cailliau]], el 12 de noviembre del 1990 publico una propuesta mas formal para constuir un sistema de hypertexto llamado WordWideWeb como una web de documentos de hypertexto para ser visualizados a traves de un [[Navegador Web]] utilizando la arquitectura [[Modelo Cliente-Servidor|Cliente-Servidor]]. Esta propuesta consideraba una red de solo lectura que se desarollaria en unos tres meses y que se necesiarian seis meses para conseguirlo. *"la creacion de nuevos vinculos y nuevos materiales para lectores <<de manera que,>> la autoria se volveria universal"*, asi como *"la notificacion automatica de los lectores cuando haya nuevo material disponible de su interes"*.

El modelo de la proouesta se habia construido utilizando DynaText, un lector de documentos SGML creado por *Electronic Book Tecnology*, una *spinn-off* del Institute for Research in Information and Scholarship de la Universidad de Brown (Providence, Rhode Island). El sistema de DynaText, del que el CERN habia adquirido licencias, era tecnicamente avanzado y tuvo un papel clave en la ampliacion del SGML ISO 8879:1986 a Hypermedia dins de HyTime (ISO 10744:1992), pero se consideraba muy caro y tenia una politica de licencias inadecuada para su uso de la comundad de fisicos de particulas de altas energias, con un impuesto por cada documento y cada modificacion.

Un ordenador NeXTcube se utilizo en el 1990 por Berners-Lee como el primer [[Servidor Web]] y tambien para escribir el primer navegador web, el [[WorldWideWeb]] (mas tarde rebajado como Nexus para evitar confusiones). En la navidad del 1990, Berners-Lee habia creado todas las herramientas necesatias para un sistema web funcional: El navegador web (que tambien era editor web), el primer servidor web y sus primeras paginas web, que describian al propio proyecto.

El 6 de agosto del 1991 publico un breve resumen del proyecto World Wide Web en el grupo de discusion `alt.hypertext` de Usenet. Esa fecha tambien marco el debut de la Web como servicio a disposicion del publico de internet, y eso que los primeros nuevos usuarios realmente se conectaron unos dias mas tarde, a partir del 23 de agosto (dia que ahora se conoce como el dia del internauta).

El concepto fundamental subyacente del hypertexto se origino en proyectos mas antiguos de la decada del 1960, como por ejemplo el Hypertext Editing System (HES) o sistema de edicion de hypertexto de la Universidad de Brown, el Proyecto Xanadu de Ted Nelson o el sistema NLS (oN-Line System) de Douglas Engelbart. Antes de todo eso, el 1945, Vannevar bush habia descrito en el ensayo "As We May Think" un dispositivo llamando "Memex (acronimo de MEMory EXpander)", que se imagino en forma de micropelicula personal para almacenar libros, enciclopedias, etc. Douglas Engelbart se inspiro en este escrito para crear el concepto de Hipervinculo.

El avance de Berners-Lee consistio en incorporar el hypertexto a Internet. En su libre *Weaving The Web (Tejiendo la red)*, explica que el sugirio repetidamente que la union entre las dos tecnologias era posible para los tecnicos, pero como nadia acepto su propuesta,  finalmente decidio por hacer su propio proyecto. Durante el proceso desarrollo un sistema de indentificadores globales unicos para los recursos del sistema web y en otros sitios: el *Universal Document Identifier* (UDI) o Identificador Universal de Documentos, que mas tarde seria conocido como [[Uniform Resource Locator]] (URL) o Localizador Uniforme de Recursos y el [[Uniform Resource Identifier]] (URI) o Identificador Unifrome de Recursos; El lenguaje de edicion y publicacion del Hyper text Markup Language ([[Índice - HTML|HTML]]) o Lenguaje de marcado de hypertexto y el Protocolo de transferencia de hypertexto ([[HTTP - Protocolos de red|HTTP]]).
